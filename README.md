# 原宇宙：無限紀元

手機優先的 AI 冒險遊戲 MVP。第一版以 Web 手遊原型呈現，使用 Phaser 建立探索場景、觸控操作、AI NPC 模擬、動態任務、隊友指令與學習型怪物。

## 啟動

```bash
npm install
npm run dev
```

本機預覽網址：

```text
http://127.0.0.1:5173/dev.html
```

## 產生可直接開啟的 HTML

```bash
npm run export:html
```

產生後可直接雙擊：

```text
index.html
```

Windows 也可以直接雙擊 `start-game.cmd`，它會啟動開發伺服器並打開遊戲網址。

不要直接雙擊 `index.html` 來啟動遊戲。這個檔案是 Vite 的入口模板，需要由 Vite 伺服器處理 TypeScript 與套件匯入。

## 已完成的 MVP 功能

- 觸控搖桿與手機 UI，可直接在第一屏開始遊玩。
- 四大區域：晨星村、北方森林、星潮海岸、失落神殿。
- AI NPC 對話模擬：可詢問神殿、怪物、村莊、世界事件，NPC 會記住對話並產生新任務。
- 動態任務系統：根據玩家行為新增任務與獎勵。
- AI 隊友璃：支援偵查、護衛、火焰支援、收集資源。
- 學習型影獸：會根據玩家左右閃避習慣改變追擊偏向。
- 星核碎片收集、威脅值、信任與聲望等持久化狀態。
- 事件掃描脈衝，可在地圖上標示高壓力世界事件。
